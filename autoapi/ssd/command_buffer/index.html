class ssd.command_buffer.CommandBuffer[source]

CommandBuffer 클래스는 명령어를 버퍼에 저장하고 최적화하는 기능을 제공합니다. 명령어는 buffer.txt 파일에 저장되며, 필요한 경우 버퍼를 비울 수 있습니다.
