      NAND 드라이버 객체

      :param value: 쓸 값 (16진수 문자열 형식)
      :type value: str



   .. py:method:: read(lba: int)

      주어진 LBA에서 값을 읽고 결과를 기록합니다.

      :type lba: int

      :type lba: int
      :param n_value: 삭제할 블록 수
